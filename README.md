# Revising Data

A personal collection of notebooks for revising core data science concepts, pandas, visualization, and general data handling. Built as a study log while working through problems and refreshing fundamentals.

## Structure

```
revising-data/
├── pandas/          # Series, DataFrame ops, merge, groupby, boolean indexing, etc.
├── matplotlib/       # Plotting and visualization notebooks
└── misc/             # Everything else | One-off exercises, notes, experiments
```

## Contents

### `pandas/`
Notebooks covering pandas fundamentals and practice problems | Series, Data Frames, Merging, Melting etc.

### `matplotlib/`
Notebooks focused on plotting basics: line/bar/scatter plots, subplots, styling, and common visualization patterns.

### `misc/`
Anything that doesn't fit neatly elsewhere | Smaller exercises, quick notes, or one-off experiments.

## Usage

Clone the repo and open any notebook with Jupyter:

```bash
git clone https://github.com/m-ahmad-amin/revising-data.git
cd revising-data
```

## Notes

This is a living, informal repo | Notebooks may be incomplete or messy, and folder contents will grow as I revisit and add new topics.
